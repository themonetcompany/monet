# AGENTS.md

## Read First
- Read `DEVELOPERS_GUIDE.md` before starting any task.

## Validation
- Run `dotnet build Monet.sln` and `dotnet test Monet.sln` before starting a task and after completing it.

## Frontend Styling
- Use Tailwind CSS for Angular styling.
- Do not add custom CSS without explicit authorization.

## Decision Records
- All project decisions (architecture, coding style, etc.) must be recorded.
- Create a decision record in the `decisions` directory for each decision.
- File name format: `decision_{yyyyMMddHHmmss}.md`
